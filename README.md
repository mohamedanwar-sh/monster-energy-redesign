# Monster Energy Redesign Concept

## Overview

A responsive Monster Energy landing page redesign concept built with React, Vite, GSAP, and Tailwind CSS. The project focuses on premium product storytelling, scroll-based motion, interactive flavor selection, and mobile-friendly presentation for a frontend portfolio.

## Features

- Animated hero experience with responsive media
- Interactive flavor and energy selection sections
- Scroll-triggered GSAP transitions
- Testimonial video cards with accessible modal playback
- Responsive navigation and footer
- Smoke tests for key user flows

## Tech Stack

- React
- Vite
- Tailwind CSS
- GSAP and ScrollTrigger
- Playwright
- ESLint

## Installation

```bash
npm install
npm run dev
```

The local development server will print the available URL in the terminal.

## Available Scripts

npm run dev
